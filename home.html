About Tasky Application

Tasky Application is the "Task Reminder Application" here, we can Plan our task by Adding it. Tasks like:
1. Birthday Reminders.
2. Important work's Reminders.
3. Resolution Reminders and
4. To do Reminders etc. According to the user's Task.
This application will help user to create their task with Title, Date, Image and Description also user can check their task by searching it on search bar and delete it after their task get completed.
